# sdkman!

Install the latest stable version of your SDK of choice (say, Java JDK) by running the following command:

$ sdk install java

Comment l'utiliser : #SDKMAN USAGE

# restaurer le réglage de la luminosité

Installer le paquet acpi-support qui permet de :

- détecter la connexion ou la déconnexion du secteur, la fermeture d'un portable et la gestion de nombreux boutons spécifiques (sur les portables Asus, Lenovo, Panasonic, Sony et Toshiba)

- suspendre, mettre en veille prolongée et redémarrer l'ordinateur, éventuellement avec les contournements indispensables à certains matériels

- modifier la luminosité de l'écran sur certains ordinateurs portables.

Download : #packages.debian.org
